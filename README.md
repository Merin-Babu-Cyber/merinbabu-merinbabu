# Modern Cloud Security Lab

This project demonstrates the deployment of a secure Microsoft 365 environment using modern cloud security tools and Zero Trust architecture.

## Tools Used

- Microsoft 365
- Azure Sentinel (SIEM/SOAR)
- Microsoft Intune
- Defender for Endpoint / Defender XDR
- Defender for Office 365
- Terraform
- Terraform Cloud
- Cloudflare Zero Trust
- Azure Security Center
- GitHub

## Key Features

- Microsoft 365 tenant deployment with Cloudflare DNS integration
- Endpoint security and compliance using Microsoft Intune
- EDR alert testing with Defender for Endpoint
- Azure Sentinel deployment using Terraform Infrastructure as Code
- Automated incident response using Sentinel playbooks
- Conditional Access and CASB security policies
- Brute force attack simulation and automated remediation
- Zero Trust security architecture

## Purpose

This lab was created to gain hands-on experience with cloud security operations, SIEM monitoring, and automated threat detection in modern enterprise environments.
